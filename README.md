# Play Store Monitor dengan Bot Telegram

Website untuk memonitor status aplikasi di Google Play Store dan mengirim notifikasi melalui Telegram ketika aplikasi dihapus.

## Fitur

- ✅ Monitor status aplikasi Play Store
- ✅ Notifikasi otomatis melalui Telegram
- ✅ Tambah/hapus aplikasi secara massal
- ✅ Konfigurasi template pesan notifikasi
- ✅ Statistik real-time
- ✅ Interface yang responsif

## Cara Penggunaan

1. Tambahkan aplikasi yang ingin dimonitor (tunggal atau massal)
2. Konfigurasi bot Telegram dengan token dan chat ID
3. Atur template pesan notifikasi
4. Sistem akan otomatis memeriksa status aplikasi setiap 30 detik
5. Dapatkan notifikasi ketika aplikasi dihapus dari Play Store

## Teknologi

- HTML5, CSS3, JavaScript (ES6+)
- Local Storage untuk penyimpanan data
- Font Awesome untuk ikon
- Responsive design
