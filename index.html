Dapatkan Chat ID

Langkah-langkah:

  1. Buka Telegram dan cari @getidsbot
  2. Start bot tersebut
  3. Tambahkan bot ke grup yang ingin dimonitor
  4. Kirim pesan /getgroupid di grup
  5. Bot akan membalas dengan Chat ID grup
